# Contributing

## Development

1. Install Node 22+ and pnpm 9+.
2. Install Python 3.12 for the worker runtime target.
3. Run `pnpm install`.
4. Start the web app with `pnpm dev:web`.

## Standards

- Use conventional commits.
- Run lint, typecheck, and tests before opening a PR.
- Keep changes aligned with `PRODUCT_REQUIREMENTS.md` and the qualification checklist.
